25. How would you implement a custom Spring Boot filter for logging incoming HTTP requests? 
      
      @Component
      public class LoggingFilter implements Filter {
          private static final Logger logger = LoggerFactory.getLogger(LoggingFilter.class);
      
          @Override
          public void doFilter(ServletRequest request, ServletResponse response, FilterChain chain)
                  throws IOException, ServletException {
              HttpServletRequest httpRequest = (HttpServletRequest) request;
              logger.info("Request Method: " + httpRequest.getMethod() + ", Request URI: " + httpRequest.getRequestURI());
              chain.doFilter(request, response);
          }
      }
26. How do you enable Swagger documentation in a Spring Boot application? 
      
      @Configuration
      @EnableSwagger2
      public class SwaggerConfig {
      
          @Bean
          public Docket api() {
              return new Docket(DocumentationType.SWAGGER_2)
                      .select()
                      .apis(RequestHandlerSelectors.basePackage("com.example"))
                      .paths(PathSelectors.any())
                      .build();
          }
      }
27. How do you secure an API in Spring Boot using JWT (JSON Web Tokens)? 
      
      @Configuration
      public class SecurityConfig extends WebSecurityConfigurerAdapter {
          
          @Override
          protected void configure(HttpSecurity http) throws Exception {
              http.csrf().disable()
                  .authorizeRequests()
                  .antMatchers("/api/**").authenticated()
                  .and()
                  .addFilter(new JWTAuthenticationFilter(authenticationManager()));
          }
      }
